# Banking System Stability Copilot

A Streamlit-based credit analyst tool for analyzing banking sector health using IMF datasets.

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Data Sources
- **FSIC**: Financial Soundness Indicators (banking metrics)
- **MFS**: Monetary and Financial Statistics  
- **WEO**: World Economic Outlook (macro data)

Place IMF CSV files in the `data/` folder.
